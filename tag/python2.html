Django 1.3 has been released
Permalink: 2011-03-23 21:38:19 by Ken Cochrane in blog tags: django python

It includes the following:

  • Class-based views
  • Logging
  • Extended static files handling
  • unittest2 support
  • Transaction context managers
  • Configurable delete-cascade
  • Contextual markers and comments for translatable strings
  • Improvements to built-in template tags
  • Caching changes
